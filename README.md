# mpchess

Multiplayer 3D chess with a Node.js server-authority backend and a browser-based Three.js client.

## Features

- Full chess rules: castling, en passant, pawn promotion, check/checkmate/stalemate detection
- Real-time multiplayer over WebSockets
- 3D rendered board with low-poly piece models

## Usage

```bash
npm install
npm start
```

Open `http://localhost:3000` in two browser tabs to play.
